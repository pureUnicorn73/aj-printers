# Treści CMS

- `pages/` zawiera aktualne strony wykorzystujące klasy `xte-*` stylowane przez
  repozytorium `baza-presta`.
- `components/` zawiera komunikaty i fragmenty wklejane w wybrane miejsca sklepu.
- `legacy/` zawiera wcześniejsze warianty stron. Nie należy ich publikować ani
  aktualizować; pozostają wyłącznie do czasu zakończenia porównania wersji.

Źródłem merytorycznym dokumentów są pliki z `../source/`.
