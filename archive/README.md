# Archiwum

Ten katalog zawiera materiały zachowane podczas porządkowania repozytorium:

- wcześniejszą paczkę wdrożeniową z 6 lipca 2026 r.;
- alternatywnie nazwane oryginały protokołów szkody;
- eksport tekstowy przeniesiony z repozytorium `baza-presta`.

Plików archiwalnych nie należy edytować ani wykorzystywać jako źródła bieżącej
publikacji. Aktualne materiały znajdują się w `content/` i `public/`.
